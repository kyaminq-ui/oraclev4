Étape 1.2 — Système de grille isométrique
✅1.2.1 — Créer la grille de combat
Concept mathématique isométrique :
Conversion Grille → World Position :
worldX = (gridX - gridY) * (tileWidth / 2)
worldY = (gridX + gridY) * (tileHeight / 4)

Conversion World → Grille :
gridX = (worldX / tileWidth + worldY / tileHeight)
gridY = (worldY / tileHeight - worldX / tileWidth)
Prompt Cursor AI :
"Crée un système de grille isométrique Unity C# avec :
- Classe GridManager (Singleton) gérant une grille NxM
- Classe Cell avec propriétés : position grille, position world, 
  isWalkable, isOccupied, occupant (GameObject)
- Méthodes : GetCell(x,y), WorldToGrid(Vector3), GridToWorld(x,y)
- Highlight de cellules avec couleurs différentes selon le type 
  (déplacement=bleu, attaque=rouge, AoE=orange)
- Utiliser ScriptableObject pour la config de la grille"

3.1.3 — Synchronisation du combat
Points critiques à synchroniser :
Architecture turn-based networking :
- Le serveur (MasterClient Photon) valide les actions
- Chaque action = RPC envoyé à tous les joueurs
- Pas de simulation côté client pour éviter les désync

Actions à synchroniser via RPC :
□ MoveCharacter(characterId, List<cellPositions>)
□ CastSpell(characterId, spellId, targetCellX, targetCellY)
□ EndTurn(characterId)
□ SelectPassive(playerId, passiveId)
□ TimerExpired(characterId)

Prompt Cursor AI :
"Crée un CombatNetworkSync Unity C# avec Photon PUN 2 :
- RPCs pour chaque action de combat
- Validation côté MasterClient avant exécution
- File d'attente d'actions pour éviter les conflits
- Resync si un joueur se reconnecte en cours de partie"

5.3.2 — Équilibrage initial des sorts
Formule de base pour les dégâts :
Dégâts = (Valeur_sort × Multiplicateur_stat) - Résistance_cible

Pour le MVP, rester simple :
- Sort de base (1 PA) : 20-30 dégâts
- Sort moyen (2 PA) : 40-60 dégâts  
- Sort puissant (3 PA) : 70-100 dégâts
- HP de base : 500-800 (pour des combats de 8-12 tours)

Ratio PA/tour : 6 PA par tour (permet 2-3 sorts par tour)
Ratio PM/tour : 4 PM par tour (déplacement significatif)
